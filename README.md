

# Test de Maîtrise du Langage Python – Projet 5 OpenClassrooms

Ce dépôt présente le livrable du Projet 5 du parcours "Développeur d’application Python" : un test technique visant à valider la synthèse des compétences acquises.

**Objectifs du projet**  
- Appliquer les connaissances en gestion de données (ETL), programmation orientée objet, design patterns (MVC)  
- Résoudre des exercices concrets dans un contexte de test technique professionnel  
- Démontrer une compréhension approfondie et critique du langage Python  

**Contenu du dépôt**  
- Solutions aux questions / exercices du test technique  
- Commentaires expliquant les choix de conception et d’implémentation  
- Respect des bonnes pratiques (PEP 8, lisibilité, modularité)  

Ce projet ne nécessite pas d’installation particulière (exécuter les scripts Python fournis si applicable).  
Il sert de preuve de validation des compétences préalables avant les projets web (Django, interfaces, etc.).



Clonez le repo et créez l'environnement virtuel.

Réalisez les exercices en suivant les instructions de chaque exercice.

Rédigez votre code dans le fichier main.py de chaque exercice.

Exécutez le code afin de vérifier qu'il fonctionne bien.

Discutez de vos exercices avec votre mentor. Il a le corrigé de tous les exercices.

Bon entraînement !
